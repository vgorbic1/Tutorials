## Text File Editors

#### Nano
```
nano file.txt
```
&uarr; &darr; page up or down

#### Vi
```
vi file.txt
```
insert/editing mode:
- i - go to edit mode (insert text).
- a - append text in edit mode.
- esc - toggle out of insert mode.
- : - before entering a command
- h j k l - cursor movement.
- x - delete character.
- dd - delete line.
- :q - quit the program.
- :w - save the file.
