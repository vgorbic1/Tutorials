## A Minimal HTML5 Document
Because all browsers (even old ones like IE6) fully support the standardized HTML5 shortcuts, we can use:
```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <title>title</title>
    <link rel="stylesheet" href="style.css">
    <script src="script.js"></script>
  </head>
  <body>
    <!-- page content -->
  </body>
</html>
```
